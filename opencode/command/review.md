---
description: Code review for quality, security, and best practices
agent: code-reviewer
---
Perform a comprehensive code review of the current changes.

Focus on:
- Code quality and best practices
- Potential bugs and edge cases
- Performance implications
- Security considerations
- Type safety and correctness
- Adherence to project conventions

Non-strict checklist:
- Are there any security concerns?
- Is error handling comprehensive?
- Are edge cases covered?
- Is the code testable? e.g. components for external services can be injected
- Are there any performance concerns for the expected scale?
- Are errors suppressed or ignored without leaving any trace?

Provide constructive feedback without making direct changes.
